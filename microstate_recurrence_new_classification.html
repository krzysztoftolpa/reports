Classification between Healthy and Schizophrenia groups using RQA values from microstates prototypes

Dataset

Microstates analysis returns a backfitted sequence of stable patterns of EEG activity on the level of the scalp. Having this sequence for each subject, one can transform it into recurrence matrix of the states in order to obtain the parameters of the recurrence quantification analysis (RQA) (Figure 1).

Figure 1: Exemplary recurrence matrix from sequence of microstates prototypes of 0-2 s interval.

For each prototype, RQA values are calculated by selecting only the occurences of one prototype and assigning the ones while zeroing the occurences of others. Thus, after the analysis 240 RQA values are obtained (16 prototypes x 15 parameters) for each of the 84 participants (45 schizophrenic and 39 healthy).

Preprocessing

Before the analysis the dataset was checked for the presence of outliers, which could potentialy bias the final results. For this purpose, IQR method was used which defines as outliers values outside:

- lower threshold: Q1 - 1.5xIQR

- upper threshold: Q3 + 1.5Ã—IQR

The identified outliers were replaced by the median value of the coresponding RQA parameter. Afterwards, the values were normalized using the minmax method which transform the values to the range: 0-1.

Group prototypes

The first step is to prepare the prototypes for each group so that we can use them as a reference for the RQA vector of each individual. The prototypes are calculated by averaging RQA values within each group.

Feature dataset

Prototypes are then compared with the individual RQA parameter values using three similarity measures:

  1. Euclidean Distance: ranges from 0 - absolute identity to some maximum possible discrepancy value

  2. Cosine Distance: ranges from 0 to 2, 0 - identical vectors, 1 - no correlation, 2 - absolutely different.

  3. Correlation: ranges from -1 to +1 with 0 implying no correlation.

The similarity values both to healthy and to schizophenic prototype are used as the features for the classification (Figure and Table 2). Note, each metric: euclidean, cosine and correlation are used seperately, one at a time.

RQA parameters selection

Additionally, the selection of the RQA parameters was performed to find values significant for the distinction between healthy and schizophrenic groups. The intuition is to select those parameters which are helpful for the further classification and discard those which introduce noise into the dataset. The following approaches were tested:

1. Absolute difference between the prototypes: prototypes for healthy and schizophrenic groups were subtracted from each other. The fraction of the highest absolute difference values were used for the similarity calculations. Here, parameters with 20% and 10% highest values were tested.

2. Statistical tests: to find the parameters values which differ significantly between healthy and schizophrenic groups, two statistical tests were used: t-test and U Mann-Whitney test. Since hundreds of parameters were tested, corrections for multiple comparisons were used: Bonferroni and Benjamini-Hochberg FDR. In total using this approach 4 possible combinations of selected parameters were tested.

3. Fisher's score: calculated as the ratio of between-group and within-group variance. A higher Fisher's Score implies the parameter is more discriminative and valuable for classification. Fraction of the parameters with the highest rank were chosen, here 20% and 10%. The implementation from skfeature.function.similarity_based was used.

4. Mutual Information: mutual information (MI) between two random variables is a non-negative value, which measures the dependency between the variables. It is equal to zero if and only if two random variables are independent, and higher values mean higher dependency. Here, the implementation from sklearn.feature_selection.mutual_info_classif which estimated the dependency with the discrete (categorical) target variable. Similarly, parameters with 20% and 10% highest MI values were tested.

After the selection of the most relevant features using one of the aforementioned methods at a time, the prototypes per schizophrenic and healthy group are created and compared with individual vectors using three similarity measures.

Classification

The models were prepared using the following classifiers:

- Logistic Regression,

- Random Forest,

- Support Vector Machine: two kernels were used - linear and rbf with default parameters.

evaluated using 5-fold crossvalidation and metrics:

- Accuracy

- ROC curve AUC

In total 120 models (10 x 3 similarity measures x 4 classifiers) were evaluated. Below are described the best classification results for each similarity metric.

The highest scores, taking into account performance of all classifiers together, were 0.873 for the ROC AUC metric and 0.834 for the accuracy, both for the correlation similarity metric. For the single classifier highest ROC AUC value was 0.876 for the SVM rbf and cosine metric while the highest accuracy was the same - 0.834. It is worth noticing the variability of the scores is low. The mean ROC AUC across all models is 0.793 with standard deviation 0.117 while the mean accuracy is 0.73 with standard deviation 0.082.
